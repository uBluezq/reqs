Requisitos

Habilidades

  • @Rush: Llevar bloques necesarios para ganar el rush, bypassear cuando se debe, alto nivel en hacer puentes, rápidos y eficaces.
  • @Defensa: Saber qué hacer con materiales, realizar una estrategia ante un ataque, compra rápida, ganar tiempo hasta recibir apoyo del compañero.
  • @Altura: Dominio de bloques, juego de herramientas (tijeras), saber utilizar escaleras y reducir su KB.
  • @Clutch: Visión ante una situación extrema, abandonar cama cuando se deba, utilizar materiales necesarios para asegurar la victoria, tener conocimiento de jugadas riesgosas pero ventajosas.

Habilidades para TierS

  • @N+: Se necesita prueba contra otro @N+.
  • @S: Se necesita tener todas las habilidades.
  • @A: Se necesita mínimo tener 3 habilidades, pueden ser cualquiera de ellas.
  • @B: Se necesita mínimo tener 2 habilidades, pueden ser cualquiera de ellas.

Si quieres realizar alguna prueba habla al MD del staff u owner.
